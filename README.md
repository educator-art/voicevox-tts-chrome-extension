# 🔊 VOICEVOX TTS Chrome拡張機能

選択したテキストをVOICEVOXで読み上げるChrome拡張機能です。

---

## 📋 必要なもの

- **VOICEVOX** がローカルにインストールされ、起動していること  
  ダウンロード: https://voicevox.hiroshiba.jp/  
  デフォルトAPIポート: `http://localhost:50021`

---

## 🚀 インストール方法

1. このフォルダをダウンロード（またはクローン）
2. `icons/` フォルダに以下のアイコン画像を用意する
   - `icon16.png`（16×16px）
   - `icon48.png`（48×48px）
   - `icon128.png`（128×128px）
3. Chromeで `chrome://extensions/` を開く
4. 右上の「デベロッパーモード」をオンにする
5. 「パッケージ化されていない拡張機能を読み込む」をクリック
6. このフォルダを選択する

---

## 📖 使い方

### 右クリックメニューから読み上げ
1. Webページ上のテキストを**マウスでドラッグして選択**
2. **右クリック**してコンテキストメニューを開く
3. **「🔊 VOICEVOXで読み上げる」** をクリック
4. ページ右下にトースト通知が表示され、読み上げが開始されます

### ポップアップ設定（拡張アイコンをクリック）
- **話者キャラクター**: 読み上げキャラを選択
- **話速**: 0.5〜2.0倍速
- **ピッチ**: 声の高さを調整
- **音量**: 音量を調整
- **テスト読み上げ**: 設定確認用

---

## 📁 ファイル構成

```
voicevox-tts-extension/
├── manifest.json   # 拡張機能の設定
├── background.js   # サービスワーカー（API通信・コンテキストメニュー）
├── content.js      # コンテンツスクリプト（音声キュー管理・順次再生）
├── popup.html      # ポップアップUI
├── popup.js        # ポップアップのロジック
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

---

## ⚠️ 注意事項

- VOICEVOXは必ず**起動した状態**で使用してください
- デフォルトのAPIポートは `50021` です。変更している場合は `background.js` と `popup.js` の `VOICEVOX_URL` を修正してください
- テキストは `。！？.!?` で文ごとに分割して順次合成・再生するため、最初の1文が合成され次第すぐ再生が始まります
